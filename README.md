 
# 🎯 Interview Slot Booking System

A Full Stack Web Application for managing interview slots.

## Tech Stack
- **Backend**: Spring Boot 3.5, Java 21, MySQL, JWT Security
- **Frontend**: React + Vite, Bootstrap, Axios

## Features
- JWT Authentication & Authorization
- Role based access (ADMIN/HR/INTERVIEWER/CANDIDATE)
- Job Listings Management
- Interview Slot Booking
- Reports & Analytics
- Dark Mode

## How to Run

### Backend
1. Create MySQL database: `interview_system`
2. Run in Eclipse: `InterviewSystemApplication.java`
3. Runs on: `http://localhost:8080`

### Frontend
1. Run: `npm install && npm run dev`
2. Open: `http://localhost:5173`
